# ZenList 📝

A modern, responsive, and lightweight To-Do Task Web Application built using **HTML, CSS, and JavaScript**. ZenList helps users efficiently organize their daily tasks while providing a fast and seamless experience through browser-based Local Storage.

## ⚙️ How It Works

1. User adds a task.
2. JavaScript dynamically updates the task list.
3. Task data is stored in the browser's Local Storage.
4. On page reload, stored tasks are retrieved automatically.
5. Users can add, delete or get task details at any time.

## 💾 Local Storage Implementation

ZenList uses the browser's Local Storage to persist task data.

### Advantages

* Faster data access
* Offline functionality
* No server costs
* Reduced complexity
* Persistent storage across sessions

### Limitations

* Data is stored only on the user's device
* No synchronization across multiple devices
* Clearing browser data removes stored tasks
* Limited storage capacity compared to databases
